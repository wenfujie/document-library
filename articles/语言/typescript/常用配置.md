## 常用配置

ts 的配置文件一般位于根目录 `tsconfig.json`

### 检索文件的路径

```js
{
	// 包含
  "include": [
    "src/**/*.ts",
    "src/**/*.d.ts",
    "src/**/*.vue",
    "types/**/*.d.ts",
    "vite.config.ts"
  ],
	// 排除
  "exclude": ["node_modules", "dist", "**/*.js"]
}

```

### 路径别名智能提示、跟踪

当在打包工具配置路径别名 alias 后，编辑器 ide 是无法识别路径提供智能提示和智能跟踪文件的。

要解决该问题，需增加如下 ts 配置，让 ts 识别到路径别名。

```ts
{
  "compilerOptions": {
    "paths": {
      "@/*": ["src/*"],
      "#/*": ["types/*"]
    }
  }
}

```
