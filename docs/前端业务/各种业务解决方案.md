## 在线查看 Excel 文件

```js
// 修改src参数为文件地址即可（必须是公网地址）
<iframe
  src="https://view.officeapps.live.com/op/view.aspx?src=https://dc-vizier-sourse.oss-cn-beijing.aliyuncs.com/test.xls"
  width="1200px"
  height="900px"
  frameborder="1"
></iframe>
```

## 模糊查询匹配文字高亮

```js
'12345'.replace(/123/, `<span style="color:red">${/123/.exec('12345')}</span>`)
```
