## 透明背景色

最后一个参数是透明度。

若只有十六进制颜色如#fff，可用[转换网站](https://www.sioe.cn/yingyong/yanse-rgb-16/)进行十六进制和 rgb 颜色相互转换。

```css
background-color: rgba(31, 52, 80, 0.5);
```

## 背景多个渐变色

`direction`

- to top（0deg/180deg）：下 → 上
- to bottom（180deg）：上 → 下
- to right（90deg）：左 → 右
- to left（270deg）：右 → 左 ‌

‌ 或 ‌

- 45deg：左上 → 右下
- -45deg：右下 → 左上 ‌

```css
background: linear-gradient(direction, red, blue, pink);
```

## 元素置灰

```css
filter: brightness(0.6);
```
